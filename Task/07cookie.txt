쿠키
	사이트에 접속할 때 생성되는 정보를 담은 4kb 이하 크기의 임시 파일을 의미
	서버에 부담을 주지 않음
	개인 정보 탈취등의 악용이 가능 해 질 수 있음
	response 객체를 사용

	연결  상태가 없는 HTTP 프로토콜을 위해 접속된 상태를 그대로 유지하고자 할 때 필요합니다. 
	쿠키는 웹 브라우저에 상태 정보를 저장하기 때문에 이후 웹 서버로 전송되는 서비스 요청에는 쿠키가 가지고 있는 정보도 함께 포함되어 전송됩니다.
	쿠키는 사용자가 저장 세션은 서버가 저장
	속도는 쿠키, 보안은 세션

response 내장 객체는 jsp 웹페이지의 실행결과를 웹브라우저로 반환해줄 때 사용하는 객체 입니다.
	웹브라우저에서 다른 페이지로 강제 이동.
	response.sendRedirect("이동할 페이지");

쿠키 생성
	쿠기를 저장할 변수의 이름 = new Cookie("cookie_name","cookie_value")

웹브라우저로 쿠키 보내기
	response.addCookie(쿠키가 저장된 변수)

쿠키 정보 가저오기
	Cookie[] 쿠키를 저장할 배열의 변수명 = request.getCookies();

쿠키의 이름&값
	쿠키를 저장한 배열의 변수.getName()
	쿠키를 저장한 배열의 변수.getValue()

쿠키 삭제
	쿠키의 유효시간을 0으로 만든다.
	setMaxAge(0(초단위))
