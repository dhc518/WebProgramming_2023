유효성 검사
	사용자가 입력한 데이터 값이 특정 규칙에 적합하게 입력되었는지를 서버에 전송하기 전에 검증하는 절차

핸들러 함수
	이베트가 발생하였을 경우 유효성 검사를 위해 매핑하는 메소드를 의미
	1) <script></script> 안에서 function 명령으로 핸들러 함수를 성성합니다. 스크립드 태그의 선언 위치는 무관합니다.
	2) 핸들러 함수 안에는 폼이름.입력항목 이름'의 속성을 지닌 객체를 생성합니다.
	3) <form> 태그 안에 name="폼이름" 속성을 성정합니다.\
	4) 폼 태그 안에서 <submit>를 누르는 이벤트가 발생하게 되면 onclick 속성을 이용하여 생성한 함수를 호출합니다.


데이터 입력 여부 검사
document.폼 이름.입력앵식 이름.value==""

데이터의 길이 확인
document.폼 이름.입력앵식 이름.value.length

숫자 여부 확인 
isNaN(document.폼 이름.입력앵식 이름.value)
