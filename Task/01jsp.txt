웹프로그래밍은 사용자가 웹 브라우저를 통해 필요한 정보를 서버에게 요청하면 
서버는 데이터베이스로 부터 그 정보를 제공하는 프로그램의 구동

2.
라우터의 역할
서로 다른 네트워크를 연결해 주는 장치를 의미합니다.

3. 
HTTP
서비스 요청과 제공을 원활하게 수행하기 위해 표준을 정한 프로토콜

HTTPS
보안 강화버전
일반 텍스트 대신 SSL이나 TLS 프로토콜을 통해 데이터를 암호화

4.
TCP/IP
TCP 데이터 전송을 위한 통신규약을 수행
IP 정보를 전달하고자 하는 최종 목적지 주소

5. 
서블릿
서블릿 규약에 따라 제작된 클래스로 서버에서 웹페이지 등등을 동적으로 동작하기위해 자바로 작성된 프로그램

6. 
웹 컨테이너
웹 서버의 컴포넌트 중 하나로 자바 서블릿과 상호작용을 수행하며 URL이요청이 올바르게 접근할 수 있도록 권한 보장
